# Inea vue project set up

## Front-end set up

- fe/

  - npm install
  - npm run dev

## Back-end set up

- be/

  - composer install
  - php -S localhost:8000

- Create a .env file in the root folder and put your variables there, like in the .env.example file.

## MySql set up

### To install db run this command

- mysql -u root -p < sql/init.sql
